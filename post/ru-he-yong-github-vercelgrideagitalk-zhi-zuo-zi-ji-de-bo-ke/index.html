如何用Github + Vercel+Gridea+Gitalk制作自己的博客

2025-04-17 2 min read

  1. 首先从 https://github.com/new 新建一个仓库，仓库名填你的 github用户名.github.io，如：lihua.github.io

  2. 到 https://github.com/settings/tokens/new 申请GitHub token，申请到的token注意保存，只显示一次，如果忘记可以刷新token，旧的也就直接作废

  3. 使用 github 账号登录 https://vercel.com，点击Add new - Project，选择刚刚创建的lihua.github.io后直接Import部署即可。部署完毕会给你一个默认域名如：lihua-github-io.vercel.app，这就是你博客的域名，此时还不能打开，因为还没有部署完毕，如果你有自己的域名，可以在setting-dimains 里添加，这里不多加叙述。

  4. 到 https://github.com/getgridea/gridea/releases 下载Gridea客户端并安装打开，点击远程进行配置，以lihua为例：

    配置填好后点击左下角“检测远程连接”，连接成功后点击保存，然后点击左下角同步。
    此时你就可以在Gridea上编写并发布你的博客啦！

  5. 如果你想让你的博客拥有评论功能，可以从Gridea-远程-评论配置 处打开并进行配置。
